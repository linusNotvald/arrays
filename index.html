Understanding Arrays in Java

Arrays in Java allow us to store multiple values of the same data type inside a single variable. They are one of the most fundamental data structures used in programming.

Basic Example

// Simple Java Array Example

public class DemoArray {

    public static void main(String[] args) {

        // Primitive array
        int[] numbers = {5, 15, 25, 35};

        System.out.print("Numbers: ");
        for (int i = 0; i < numbers.length; i++) {
            System.out.print(numbers[i] + " ");
        }

        System.out.println();

        // Object array
        String[] cities = {"Delhi", "Mumbai", "Chennai"};

        System.out.print("Cities: ");
        for (int i = 0; i < cities.length; i++) {
            System.out.print(cities[i] + " ");
        }
    }
}

Output

Numbers: 5 15 25 35
Cities: Delhi Mumbai Chennai

Declaring and Creating an Array

int[] arr;        // Declaration
arr = new int[4]; // Memory allocation

Once created, the array size is fixed and cannot be changed.

Using Array Literal

int[] marks = {80, 85, 90, 95};

Java automatically determines the size of the array.

Accessing Elements

int[] data = {10, 20, 30};

System.out.println(data[0]);  // 10
System.out.println(data[2]);  // 30

Valid index range: 0 to data.length - 1

Updating Elements

int[] data = {10, 20, 30};

data[1] = 50;
System.out.println(data[1]);   // 50

Traversing an Array

int[] values = {1, 2, 3, 4, 5};

for (int i = 0; i < values.length; i++) {
    System.out.print(values[i] + " ");
}

Passing Array to a Method

public static void printSum(int[] arr) {

    int total = 0;

    for (int i = 0; i < arr.length; i++) {
        total += arr[i];
    }

    System.out.println("Total: " + total);
}

Real Life Examples of Arrays

  • Student Marks: Storing marks of multiple subjects
  • Temperature Records: Daily weather data
  • Roll Numbers: List of students in a class
  • Monthly Expenses: Expense tracking for 12 months
  • Product Prices: Prices in a shopping cart

Common Error

int[] arr = new int[3];
System.out.println(arr[4]);   // ArrayIndexOutOfBoundsException

Advantages of Arrays

  • Fast access using index (O(1))
  • Simple structure
  • Efficient memory usage

Limitations of Arrays

  • Fixed size after creation
  • Stores only one data type
  • Insertion and deletion are costly

Summary

Arrays help organize related data efficiently and provide constant-time access using indexes. They are widely used in algorithms and real-world applications.
