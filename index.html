Understanding Arrays in Java

Arrays in Java allow us to store multiple values of the same data type inside a single variable. They are one of the most fundamental data structures used in programming.

Basic Example

// Simple Java Array Example

public class DemoArray {

    public static void main(String[] args) {

        // Primitive array
        int[] numbers = {5, 15, 25, 35};

        System.out.print("Numbers: ");
        for (int i = 0; i < numbers.length; i++) {
            System.out.print(numbers[i] + " ");
        }

        System.out.println();

        // Object array
        String[] cities = {"Delhi", "Mumbai", "Chennai"};

        System.out.print("Cities: ");
        for (int i = 0; i < cities.length; i++) {
            System.out.print(cities[i] + " ");
        }
    }
}

Output

Numbers: 5 15 25 35
Cities: Delhi Mumbai Chennai

Explanation

Concept Meaning
Array Collection of similar data stored in continuous memory
Index Position number of element (starts from 0)
length Property used to get total number of elements
Loop Used to access elements one by one

Declaring and Creating an Array

int[] arr;        // Declaration
arr = new int[4]; // Memory allocation

Once created, the array size cannot be changed.

Using Array Literal

int[] marks = {80, 85, 90, 95};

Java automatically determines the size based on the number of elements.

Accessing Elements

int[] data = {10, 20, 30};

System.out.println(data[0]);  // 10
System.out.println(data[2]);  // 30

Valid index range: 0 to data.length - 1

Updating Elements

int[] data = {10, 20, 30};

data[1] = 50;
System.out.println(data[1]);   // 50

Traversing an Array

int[] values = {1, 2, 3, 4, 5};

for (int i = 0; i < values.length; i++) {
    System.out.print(values[i] + " ");
}

Passing Array to a Method

public static void printSum(int[] arr) {

    int total = 0;

    for (int i = 0; i < arr.length; i++) {
        total += arr[i];
    }

    System.out.println("Total: " + total);
}

Real Life Examples of Arrays

  • Student Marks: Storing marks of 5 subjects in an array like {85, 90, 78, 88, 92}
  • Temperature Records: Storing daily temperatures of a week
  • Roll Numbers: Storing roll numbers of students in a classroom
  • Monthly Expenses: Tracking expenses for 12 months using an array
  • Product Prices: Storing prices of items in a shopping cart

In real applications like banking systems, exam result systems, and inventory management, arrays are frequently used to store and process structured data efficiently.

Common Error

int[] arr = new int[3];
System.out.println(arr[4]);   // Causes ArrayIndexOutOfBoundsException

Advantages of Arrays

  • Fast element access using index
  • Simple and easy to use
  • Efficient memory structure

Limitations of Arrays

  • Fixed size after creation
  • Can store only one data type
  • Insertion and deletion operations are costly

Summary

Arrays are essential in Java programming. They help organize related data efficiently and provide constant-time access using indexes. They are widely used in algorithms, data processing, and real-world applications.
