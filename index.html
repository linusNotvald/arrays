Example Python Code

# Basic Python Program

# Taking input
name = input("Enter your name: ")
age = int(input("Enter your age: "))

# Printing output
print("Hello", name)

# Condition check
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

# Loop example
print("Numbers from 1 to 5:")
for i in range(1, 6):
    print(i)

Code Explanation

Concept Description
Input Reads data from the user
Variables Stores user data
Condition Checks age eligibility
Loop Repeats printing numbers

How the Program Works

  1. User enters their name and age
  2. The program greets the user
  3. It checks if the user is an adult
  4. It prints numbers from 1 to 5
